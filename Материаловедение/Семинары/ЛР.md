# Твердость
> [!definition] Твердость
> **Твердость** — способность материала сопротивляться внедрению в него другого материала, которой в измерительном приборе будет называться **индентор**.

## Метод измерения твердости по Бринеллю (HB)

Измеряется в $\left( \text{HB} \right)$

Индентор — стальной закаленный шарик.

## Метод измерения твердости по Роквеллу 

Существуют следующие шкалы:
- $\text{HRB}$
	Индентор — стальной закаленный шарик.
	Нагрузка $100~\text{кгс}$.
- $\text{HRC}$
	Индентор — алмазный конус $\left( 120\degree \right)$.
	Нагрузка $150~\text{кгс}$.
- $\text{HRA}$
	Индентор — алмазный конус $\left( 120\degree \right)$.
	Нагрузка $100~\text{кгс}$.

## Метод измерения твердости по Виккерсу (HV)

Индентор — алмазная пирамида.

Применяется, когда важна точность измерений.
